### Install on my zsh
+ cd zsh folder then run below shell:
```shell
	./install-on-zsh.sh
```
+ install zsh-syntax-highlighting and zsh-autosuggestions plugins
```shell
	./install-plugins.sh
```

### Get vim and zsh configsa
+ cd configs floder then:
```
./write-configs.sh
```
### source 
```
source ~/.zshrc
```
